# Roadmap

Future plans and feature roadmap for Nexus Launcher.

## 🚀 Planned Features

### 🤖 Direct AI Responses
- ✨ Get instant answers from LLMs directly within the popup without opening a browser.
- 🔌 **OpenRouter Integration**: Access 100+ free and paid models (Llama 3, Mistral, Qwen) via a single API.
- ♊ **Google Gemini Support**: Native integration for lightning-fast reasoning using Gemini 1.5 Flash/Pro.

### 👥 Multi-Profile Support
- 📂 Create different provider sets for Work, Research, and Entertainment.

### 🌐 Community & Ecosystem
- 🎨 **Theme Marketplace**: Share and discover custom themes from the community.
- 🛠️ **Provider Plugin System**: Allow developers to create complex data parsers for different websites.

### ⚡ Action-Based Features
- 📂 **File Operations**: Open files, folders, or documents directly from selected paths
- 🖥️ **Script Execution**: Run PowerShell, Python, or Batch scripts with selected text as input
- 🚀 **App Launcher**: Launch applications with selected text as parameters (e.g., open VS Code with file path)
- 📋 **Clipboard Actions**: Copy, transform, or manipulate selected text (uppercase, lowercase, encode/decode, save to file)
- 🔗 **URL Actions**: download files from URLs, shorten links, or extract domain names from selected text

### 🎯 Smart Features
- 📝 **Text Transformations**: Convert text formats (Markdown → HTML, JSON → YAML, etc.)
- 🔍 **Quick Calculations**: Evaluate math expressions directly in the popup
- 🌍 **Translation**: Instant translation using Google Translate or DeepL API
- 📊 **Data Extraction**: Parse and extract structured data from selected text (emails, phone numbers, dates)

### 🖥️ Cross-Platform Expansion
- 🍎 **macOS Support**: Port to macOS using native hotkey system (Karabiner-Elements or Hammerspoon)
- 🐧 **Linux Support**: Implement using X11/Wayland global hotkeys

### 🎨 UI/UX Enhancements
- 🌙 **Auto Dark Mode**: Sync with system theme automatically
- ⌨️ **Keyboard Navigation**: Full keyboard control with arrow keys and shortcuts
- 🔊 **Sound Effects**: Optional audio feedback for actions (toggle on/off)
- 📌 **Pinned Providers**: Pin frequently used providers to the top
- 🔄 **Recent Searches**: Quick access to recent search queries
